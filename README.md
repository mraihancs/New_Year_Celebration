🎆 **New Year - Celebration Landing Page**

New Year is a visually striking, responsive landing page designed to capture the festive spirit of the holiday season. Built with HTML and Tailwind CSS, the project focuses on high-impact typography, modern layouts, and a vibrant color palette to create an engaging user experience.

🌟 Key Features
Festive Hero Section: A bold, modern header featuring customized New Year imagery and welcoming typography.

Responsive Grid Layout: Utilizes Tailwind's flex and grid utilities to ensure the celebration cards and gallery look perfect on mobile and desktop.

Optimized Performance: A lightweight, purely frontend project with no heavy dependencies for lightning-fast load times.

Modern Aesthetics: Includes clean hover effects, rounded containers, and high-contrast text for a premium look.

Mobile-First Approach: Fully optimized for handheld devices to share the holiday cheer on the go.

💻 Technology Stack
Structure: HTML5

Styling: Tailwind CSS (Utility-first framework)

Assets: Optimized Images and Google Fonts
